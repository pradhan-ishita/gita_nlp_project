About This Project

Project Title

NLP-based Analysis of Bhagavad Gita Verses

Objective

This project applies Natural Language Processing techniques to Bhagavad Gita verses in order to recommend spiritually relevant verses based on human emotions and detect major themes across chapters.

Modules

  • Emotion-based Verse Recommender
  • Theme Detection Across Chapters
  • Emotion Interpretation and Query Expansion

Technologies Used

  • Python
  • Flask
  • Pandas
  • Scikit-learn
  • NLTK
  • HTML/CSS

Future Scope

  • Sentence-BERT semantic retrieval
  • Voice-based input
  • Multilingual support
  • User accounts and saved favorites
